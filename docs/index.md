# Open WebUI Documentation

Welcome to the comprehensive documentation for Open WebUI, an extensible, feature-rich, and user-friendly self-hosted AI platform designed to operate entirely offline.

## Overview

Open WebUI is a powerful platform that supports various LLM runners like Ollama and OpenAI-compatible APIs, with built-in inference engine for RAG, making it a complete AI deployment solution.
